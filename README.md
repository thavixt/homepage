# Your home page

Set the `https://homepage.komlosidev.net` URL as your home page, and customize it to you liking!

## Features:

- [ ] calendar
- [ ] light/dark theme
- [ ] local settings
- [ ] localization with react-i18n
- [ ] notes
- [ ] timer/reminder
- [x] bookmarks with import/export
- [x] dynamic background
- [x] site statistics
- [x] todo list
- [x] weather widget

### Made with

- React + Typescript
- React Router
- Redux
- Tailwind
- Vite
- shadcn components
- deployed with Vercel

# Your home page

Set the `homepage.komlosidev.net` URL as your home page, and customize it to you liking!

Features:
- [x] bookmarks
- [x] showing local weather
- [x] todo list
- [x] dynamic background
- [ ] local settings
- [ ] notes
- [ ] calendar
- [ ] site statistics

### Made with

- React + Typescript
- shadcn components
- React-router
- React-redux
- Tailwind
- Vite
- deployed on Vercel
